# Patient-care-analysis-dashboard
A data analysis project exploring hospital patient data, treatment cost, medical conditions, and healthcare outcomes using a visual dashboard.
This project focuses on analyzing a hospital dataset to gain insights into patient demographics, treatment procedures, healthcare costs, and patient outcomes. The dataset includes variables such as age, gender, medical condition, procedure type, cost of treatment, length of hospital stay, readmission status, patient outcome, and satisfaction level.


The aim of the analysis is to identify patterns and relationships within the data that can help improve healthcare service delivery, reduce hospital readmission rates, and enhance patient satisfaction. Using data analysis tools such as Excel, the project evaluates how different factors influence treatment outcomes and overall hospital performance.
Tools Used- Microsoft Excel


**Key Insights**
- Patient demographics distribution
- Treatment cost analysis
- Hospital stay duration
- Patient satisfaction levels
